คุณคือผู้เชี่ยวชาญด้าน UI/UX และ Front-end Developer ยุคใหม่ 
ช่วยเขียนโค้ด HTML + Tailwind CSS (ใช้ผ่าน CDN) สำหรับสร้างเว็บไซต์ Portfolio หน้าเดียว (Single Page Portfolio) 
สำหรับนักเรียนชั้น ม.3 ที่ใช้สำหรับยื่นเข้าศึกษาต่อระดับชั้น ม.4 

โดยการออกแบบและสร้างเว็บไซต์นี้จะอ้างอิงตามหลักการ 5W1H ดังนี้:

1. WHAT (สร้างอะไร): 
- เว็บไซต์ Portfolio แนะนำตัวของนักเรียน ม.3 เพื่อยื่นเข้า ม.4 สายการเรียน [ระบุสายเรียน เช่น วิทย์-คณิต / ศิลป์-ภาษา / คอมพิวเตอร์]
- ชื่อนักเรียน: [ใส่ชื่อ-นามสกุลของคุณ]

2. FEATURES (ฟีเจอร์ที่ต้องมี 3-5 อย่าง):
- Section 1: Hero & Intro (หน้าแรกแนะนำตัวแบบกระชับ มีปุ่มกดดูผลงาน)
- Section 2: Education & GPAX (ประวัติการศึกษาและเกรดเฉลี่ยที่โดดเด่น)
- Section 3: Activities & Projects (กล่องแสดงภาพกิจกรรม ผลงานเด่น หรือจิตสาธารณะ พร้อมคำอธิบายสั้นๆ)
- Section 4: Future Goals & Contact (เป้าหมายในอนาคตว่าทำไมอยากเรียนสายนี้ และช่องทางติดต่อ)

3. WHO (ใครใช้/ใครดู):
- คณะกรรมการและอาจารย์ผู้คัดเลือกเข้า ม.4 (ต้องการเว็บที่อ่านง่าย โหลดไว ข้อมูลชัดเจน ไม่ซับซ้อน)

4. LOOK (หน้าตา/ธีมสี):
- สไตล์: Modern Minimalist ดูสะอาดตา น่าเชื่อถือ แต่ยังคงความสดใสของวัยรุ่น
- โทนสี: ใช้สี [ระบุสี เช่น น้ำเงิน-ฟ้า / ส้ม-เหลือง / เขียว] เป็นสีหลัก (Primary Color)
- Layout: เป็น Responsive Design รองรับการเปิดทั้งบนคอมพิวเตอร์และมือถืออย่างสวยงาม 
- ฟอนต์: ภาษาไทยที่อ่านง่ายและทันสมัย (เช่น Prompt หรือ Sarabun จาก Google Fonts)

5. WHY (ทำไมต้องสร้าง):
- เพื่อสร้างความแตกต่างและโดดเด่นจากพอร์ตโฟลิโอรูปเล่มกระดาษแบบเดิมๆ แสดงให้เห็นถึงทักษะด้านเทคโนโลยีและการจัดการข้อมูลที่เป็นระบบ

6. HOW (แนวทางการ Implement และข้อจำกัด):
- เขียนด้วย HTML และ Tailwind CSS รวมอยู่ในไฟล์เดียว (Single File) เพื่อให้ก๊อปปี้ไปเปิดใช้งานได้ทันที
- สิ่งที่ไม่ต้องการ: แอนิเมชันที่ฉูดฉาดหรือช้าเกินไป, เอฟเฟกต์ที่ดูรกสายตา และโค้ดที่ซับซ้อนเกินจำเป็น

ช่วยเจนเนอเรตโค้ด HTML ที่สมบูรณ์ สวยงาม พร้อมใส่ Placeholder ของรูปภาพ (เช่น จาก unsplash) และข้อความตัวอย่างที่สามารถนำไปแก้ไขเองได้ง่ายๆ มาให้หน่อยครับ
